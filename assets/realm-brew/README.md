# Realm Brew Asset Setup

The Realm Brew Complete Bundle v0.2 KS provides hand-drawn hex tiles and overlays for enhanced Dungeon/Cavern themes. These assets are optional — without them, the Dungeon theme uses procedural (Canvas-drawn) tiles.

## Setup

1. Obtain the **Realm Brew - Complete Bundle v0.2 KS** from your Kickstarter purchase
2. Copy the contents into this directory with the following structure:

```
assets/realm-brew/
├── tiles/
│   ├── man-hewn-dungeons/          ← Copy from "0 • Realm Brew - Man Hewn Dungeons Tiles/"
│   ├── subterranean-rivers/        ← Copy from "0 • Realm Brew Subterranean Rivers - Digital Tiles/"
│   └── underdark-caverns/          ← Copy from "0 • Realm Brew Underdark Caverns - Digital Tiles/"
├── overlays/
│   ├── man-hewn-dungeons/          ← Copy from "0 • Realm Brew - Man Hewn Dungeons Overlays/"
│   ├── subterranean-rivers/        ← Copy from "0 • Realm Brew Subterranean Rivers - Digital Overlays/"
│   ├── underdark-caverns/          ← Copy from "0 • Realm Brew Underdark Caverns - Digital Overlays/"
│   ├── alchemists-workshop/        ← Copy from "1 • Realm Brew - Alchemists' Workshop Overlays/"
│   ├── goblins-hideout/            ← Copy from "1 • Realm Brew - Goblins' Hideout Overlays/"
│   └── red-dragons-lair/           ← Copy from "1 • Realm Brew - Red Dragon's Lair Overlays/"
└── README.md                       (this file)
```

3. The game will automatically detect the assets and enable enhanced tile options for the Dungeon theme.

## Asset Details

- **Tiles:** 111 hex-shaped PNG tiles (1200×1039px each), 3 themes
- **Overlays:** 292 transparent PNG overlays (variable sizes), 6 packs
- **Total size:** ~296MB
